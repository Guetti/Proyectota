# Contribuir

Gracias por tu interés en contribuir.

## Requisitos

- Node.js 20+
- pnpm 8+

## Cómo contribuir

1. Haz un fork y clona tu fork.
2. Instala dependencias:
   - `pnpm install`
3. Crea una rama:
   - `git checkout -b feat/tu-cambio`
4. Ejecuta el proyecto en local:
   - `pnpm dev`
5. Realiza tus cambios con commits pequeños y claros.
6. Verifica calidad:
   - `pnpm lint`
7. Haz push a tu fork y abre un Pull Request.

## Estilo de código

- Respeta el formato existente.
- Evita cambios no relacionados.
- Mantén los componentes y estilos consistentes con el diseño actual.

## Mensajes de commit

Usa un formato simple:

- `feat: agrega hero`
- `fix: corrige typo en about`
- `docs: actualiza README`

## Pull Requests

Incluye en el PR:

- Qué mejora aporta.
- Capturas si hay cambios visuales.

## Reportar bugs o pedir features

Usa los Issues con:

- Pasos para reproducir.
- Resultado esperado vs real.
- Capturas o logs si ayuda.

## Código de conducta

Al contribuir, aceptas mantener un trato respetuoso.
